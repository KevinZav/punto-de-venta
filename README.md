# punto-de-venta

## Requires
```
nvm install 18.16.0
nvm use
```

Then, when we have node version installed:
```
npm install --global yarn
npm install --global @vue/cli
```

Finally:
```
yarn install
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
